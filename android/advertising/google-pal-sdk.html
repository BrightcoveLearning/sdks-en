Google PAL SDK

As of v2.6.0 the SSAI plugin facilitates usage of Googleâ€™s PAL SDK with a simple API. If PAL SDK is enabled, the plugin will load in the SDK script from Google and provide a set of default parameters to generate an identifier that will be appended to the ad request for VOD and Live content. You can customize the parameters on a per-player basis or at runtime before a video source is set.

Setting up the Ad Tag

The SSAI plugin will append a givn parameter to requests for VMAP (VOD) or M3U8 manifests (Live). This can be seen by inspecting the requests in the Network tab of the browsers dev tools: (imagen aqui)
