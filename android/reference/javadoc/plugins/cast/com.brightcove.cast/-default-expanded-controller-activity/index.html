open class DefaultExpandedControllerActivity

The default Expanded Controller Activity provided for the Cast plugin. It sets the default buttons for the controller, it changes the progress bar to match the default Brightcove Media Controller progress bar, and attempts to set the background image scale type to DEFAULT_SCALE_TYPE.
