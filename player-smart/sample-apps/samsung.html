Overview

This example plays a video returned from the Brightcove Playback API, which is the latest and recommended API to retrieve content from your Video Cloud library.

For this option, you will need a Policy Key. If you are not familiar with Policy Keys, see Policy API Overview.

Prerequisites

  • Tizen Studio: Ensure you have Tizen Studio installed on your system. You can download it from here.

Create Certificates

Before you can run your application on a Tizen device, you need to create a certificate profile. Follow these steps:

  1. Launch Tizen Studio.

  2. Navigate to Tools > Certificate Manager.

  3. Create a new certificate profile. According to the Tizen policy, all applications must be signed with valid certificates before they can be installed or run. The signature verifies the source of the application and makes sure it has not been tampered with since its publication. Detailed instructions can be found here.

Launching the Application

Download the sample

  1. Clone or download the Web & Smart TV SDK samples to your local system.

  2. Navigate to the sdk/packages/web-sdk-vanilla-demo-app/tizen/ sample app.

  3. Run the npm install command.

  4. Open and run the newly created workspace.

Enable Developer Mode

  1. For physical devices navigate to the Apps panel on your Samsung TV.

  2. Input the code 12345 using the remote control to enable developer mode.

  3. Insert the IP address of your laptop (A restart is required after enabling developer mode.).

Connect Tizen Studio to the Device

  1. In Tizen Studio, navigate to Tools > Device Manager

  2. In the Device Manager, click Remote Device Manager in the top right corner to manage devices.

  3. Add a New Device:

    • Click scan to search for available devices.

    • Or click to manually insert the IP address of the TV you want to connect to.

  4. Once connected, close the Device Manager. Here, For more details.

Import Project into Tizen Studio

  1. In Tizen Studio navigate to File > Import > General > Existing Projects into Workspace.

  2. Select the directory containing the Tizen project.

  3. Right-click on the project and select Run As > Tizen Web Application (for physical devices) or Tizen Web Simulator Application (Samsung TV) for the simulator.

Modify the Project for Simple Video Playback

Edit the index.tizen.html file to include a simple video player. Here's an example modification:

<html>
<head>
    <title>Simple Video App</title>
</head>
<body>
    <video-js id="my-player" data-video-id="your-video-id" controls></video-js>
    <script>
        videojs('my-player');
    </script>
</body>
</html>
