Overview

Clien-Side allows for the easy integration of video advertising into your media players built with the Brightcove Web & Smart TV SDK. The IMA3 integration facilitates pre-roll, mid-roll, and post-roll ad placements, as well as sophisticated ad management capabilities.

Prerequisites

  • A Brightcove account with access to the Video Cloud.

  • The Brightcove Web & Smart TV SDK installed in your project.

  • Access to the Google IMA SDK resources and an Ad Manager account.

Integration

  1. Import the Player (with UI) class from the Brightcove SDK.

    import { Player, IntegrationsManager } from '@brightcove/web-sdk/ui';

  2. Import the ImaClientSide integration class from the Brightcove SDK.

    import { ImaClientSideIntegrationFactory } from '@brightcove/web-sdk/integrations/imaClientSide';

  3. Import the ImaClientSide integration CSS from the Brightcove SDK.

    import '@brightcove/web-sdk/integrations/imaClientSide/styles';

  4. Setup ima-client-side integration.

    IntegrationsManager.registerImaClientSideIntegrationFactory(ImaClientSideIntegrationFactory);

  5. Create and Configure the Player.

    const player = new Player({accountId:'<account-id>'});

 player.updateConfiguration({
   integrations: {
     imaClientSide: {
       serverUrl: '',
       requestMode: 'onload',
       hardTiemouts: false,
       // etc
     },
   },
 });

  6. Attach the player to a mount root (DOM element in the page):

    const root = document.querySelector('#player-mount-root');
 player.attach(root);

  7. Listen to the SdkLoaded event and configure the ad integration using your specific ad tag URL provided by Google Ad Manager.

    player.one(Player.Event.Integration.ImaClientSide.SdkLoaded, ({ imaClientSideSdk }) => {
  const adTag = 'your-ad-tag-url';
  player.integrations.imaClientSide.adRequest(adTag);
});

Configuration Options

This section provides detailed information about the various configuration options available for Client-Side Ad Insertion (CSAI) when using the Web & Smart TV SDK.

CSAI Configuration Options
Configuration Option Type Description Default
sdkUrl string The URL to load the IMA3 SDK. `${protocol}//imasdk.googleapis.com/js/sdkloader/ima3.js`
serverUrl string URL of the ad server from which ads will be requested. `'${protocol}//pubads.g.doubleclick.net/gampad/ads?sz=400x300&iu=%2F6062%2Fiab_vast_samples&ciu_szs=300x250%2C728x90&gdfp_req=1&env=vp&output=xml_vast2&unviewed_position_start=1&url=[referrer_url]&correlator=[timestamp]&cust_params=iab_vast_samples%3Dlinear'`
requestMode 'onload' | 'onplay' | 'ondemand' | 'once' Defines when the ad requests should be made. 'onload'
hardTimeouts boolean If set to true, ads that take too long to load will be abandoned. true
showVpaidControls boolean Determines whether custom player controls for VPAID ads should be displayed. false
clickTrackingElement HTMLElement | undefined Specifies the alternative video ad click element. If undefined, the IMA SDK handles clicks. undefined
postRollTimeout number Timeout setting for post-roll ads in videojs-contrib-ads. Overrides the default timeout value. same as timeout
preRollTimeout number Timeout setting for pre-roll ads in videojs-contrib-ads. Overrides the default timeout value. same as timeout
useMediaCuePoints boolean Enables the use of Video Cloud ad cue points to trigger ads. false
vpaidMode string Specifies the VPAID 2 mode for the IMA HTML5 SDK. Defaults to the SDK's current setting, which is ENABLED. undefined

API Methods Summary

CSAI API Methods
Method Description Parameters Parameter Type Returns
adDisplayContainer Returns the container where ads are displayed. None N/A AdDisplayContainer | null
adMacroReplacement Replaces macros in the ad URLs. url string string
adsLoader Returns the AdsLoader instance responsible for loading the ads. None N/A AdsLoader | null
adsManager Returns the AdsManager instance which manages the playback of ads. None N/A AdsManager | null
adRequest Makes an ad request using the provided ad request URL. adRequestUrl string void
currentAd Returns the currently playing ad. None N/A Ad | null
isVpaid Checks if the current ad is a VPAID (Video Player Ad-Serving Interface Definition) ad. None N/A boolean
setAdsRenderingSettings Sets the rendering settings for ads. adsRenderingSettings AdsRenderingSettings void
